# React可视化设计器
基于Formily表单驱动
* 💪 React 17.x
* 🔥 [Formily 2.x](https://formilyjs.org/zh-CN)
* 🔥 TypeScript 4.x
* 🔥 ECharts 5.x

### 预览

> 若无法访问，表示当前网络不支持ipv6

![截图](https://repository-images.githubusercontent.com/452603611/62ec66d4-fdb1-4179-9a73-c1576502dbfd)

## 安装与使用

```bash
# 安装依赖
npm install
# 跑起来！
npm run start
# 构建发布
npm run build
```

```bash

|    └── schema           # 公共schema
|    └── widgets          # 组件目录
└── pages
    └── Design           # 设计器页面
```
