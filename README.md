# React可视化设计器
基于Formily驱动表单
* 💪 React 17.x
* 🔥 TypeScript 4.x
* 🔥 ECharts 5.x

## 安装与使用

```bash
# 安装依赖
npm install
```

```bash
# 跑起来！
npm run start
```
```bash
# 构建发布
npm run build
```
